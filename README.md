Курлык! Kurlyk!

## Сборка проекта
- `yarn` - установка зависимостей  
- `yarn run server:build` - сборка бэка

## Запуск
- `yarn run server:start` - запуск бэка  
- `yarn run dev` - запуск фронта(на 127.0.0.1)
